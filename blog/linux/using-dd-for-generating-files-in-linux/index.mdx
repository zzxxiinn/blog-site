## 这篇文章讲什么？
在服务端开发过程中，经常遇到需要特定大小的文件的需求，查了一下，linux 环境下有 dd 这个命令，可以快速打到这个效果。在这里记录一下 dd 的一些使用方式和技巧。

## 正文

### dd 命令是是什么

Linux dd 命令用于读取、转换和输出数据。可以从 std-in 或者文件中读取数据，根据指定的格式转换数据，然后输出都文件或者 std-out 中。

### 常用参数
- if=your-file-name: 输入文件名，默认为 std-in
- of=your-file-name: 输出文件名，默认为标准输出
- ibs=bytes: 一次读入 bytes 个字节
- obs=bytes: 一次输出 bytes 个字节
- bs=bytes: 同时设置读入/输出 bytes 个字节
- count=blocks: 仅拷贝 blocks 个块，块大小等于 ibs


### 示例
```bash
# 当前目录生成一个名为 gigabyte 大小为 1G 的文件
$ dd if=/dev/zero of=gigabyte bs=1M count=1000 

# 当前目录生成一个名为 megabyte 大小为 1M 的文件
$ dd if=/dev/zero of=megabyte bs=1M count=1

# 当前目录生成一个名为 kilobyte 大小为 1K 的文件
$ dd if=/dev/zero of=kilobyte bs=1K count=1

# 将tmp目录下的大文件写0，可以释放磁盘空间
$ dd if=/dev/zero of=/tmp/bigfile
```
