## E1: Find First and Last Position of Element in Sorted Array

题为：找一个数字在数组中的范围
```text
Given an array of integers nums sorted in **non-decreasing** order, find the starting and ending position of a given target value.

If target is not found in the array, return [-1, -1].

You must write an algorithm with O(log n) runtime complexity
```


```python
class Solution:
    def searchRange(self, nums: List[int], target: int) -> List[int]:
        left = 0
        right = len(nums) - 1

        start = -1
        end = -1

        if len(nums) == 0:
            return [-1, -1]

        # get left bound
        while left <= right:
            mid = (left + right) // 2
            if nums[mid] < target:
                left = mid + 1
            else:
                right = mid -1
        if left >= len(nums):
            return [-1, -1]

        if nums[left] == target:
            right = len(nums) - 1
            start = left
            end = left

            # get right bound
            while left <= right:
                mid = (left + right) // 2
                if nums[mid] > target:
                    right = mid - 1
                else:
                    left = mid + 1

            if nums[right] == target:
                end = right

        return [start, end]
```
